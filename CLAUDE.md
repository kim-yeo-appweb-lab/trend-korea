# CLAUDE.md

## 프로젝트 개요

**트렌드 코리아** — 대한민국 사회 이슈/사건을 추적/분석하는 서비스.

| 디렉토리           | 역할       | 기술 스택                                  | 상세                             |
| ------------------ | ---------- | ------------------------------------------ | -------------------------------- |
| `trend-korea-api/` | 백엔드 API | FastAPI + SQLAlchemy 2.0 + PostgreSQL (uv) | `trend-korea-api/CLAUDE.md` 참조 |
| `trend-korea-app/` | 프론트엔드 | Next.js 16 + React 19 + TypeScript (pnpm)  | `trend-korea-app/CLAUDE.md` 참조 |

각 프로젝트는 독립된 GitHub 레포지토리이며, 자체 CLAUDE.md에 상세 정보가 있습니다.

## 상세 문서 (필요시 참조)

- `docs/PRD.md` — 통합 제품 요구사항 (프론트엔드 + 백엔드 + 어드민)
- `docs/ROADMAP.md` — 개발 로드맵 (Phase별 태스크, 완료 상태 추적)
- `docs/LEANCANVAS.md` — 린 캔버스 (문제-솔루션-가치제안-고객-경쟁우위-지표-수익)

## 도메인 용어

| 한글   | 영문 (코드) | 설명                              |
| ------ | ----------- | --------------------------------- |
| 사건   | `Event`     | 특정 일자에 발생한 단일 이벤트    |
| 이슈   | `Issue`     | 언론/SNS에서 지속 추적되는 주제   |
| 트리거 | `Trigger`   | 이슈에 대한 새로운 업데이트       |
| 태그   | `Tag`       | 사건/이슈 분류 라벨               |
| 출처   | `Source`    | 뉴스 기사, 공식 발표 등 참고 자료 |
